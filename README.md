Adkintun Mobile Server
======================

Adkintun Mobile server.

* VirtualEnv
* Postgresql
* SQLAlchemy.
* Flask-SQLAlchemy.
* Flask-Migrate.
* Flask-RESTful.
* Flask-Admin.
